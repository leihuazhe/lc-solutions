# 总结

1.子集/组合问题,因为需要考虑不能回溯已经回溯的内容,所以需要使用 start 来标识从哪个 position 开始。
2.排列问题,不需要考虑从哪个pos开始，所以不需要 start, 但是要考虑不能回溯已回溯的,所以需要 used 来标识已访问到. (1,2,3 访问了1后，不能再访问1)
3.剪枝：剪枝问题需要先排序数组。
